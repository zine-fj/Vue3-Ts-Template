# VUE3+Ts 

## intro
vue3+Ts 基础模板，即开即用。


## 环境依赖
`node 14+`、`vueCLi 4+`

## 项目运行
- 安装
```js
npm install
// or 
yarn
```
- 运行
```js
// 开发环境（默认模式时 development）
npm run serve
```
- 打包
```js
// 生产包（默认模式时 production）
npm run build
```
- 环境区分
```js
development: 开发
test: 测试
production: 生产
```
